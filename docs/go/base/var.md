# 变量

## 变量的声明方式

##### 方式一、简写（常用）

```go
e := 10
```

注意：这种方式不支持全局变量声明

##### 方式二、声明并赋值

```go
var b string = "rjr"
```

##### 方式三、声明并赋值（不声明数据类型）

```go
var c = false
```

##### 方式四、只声明不赋值

```go
var a init
```

## 多变量声明

##### 单行写法

```go
var a,b int = 100,200
var c,d = 300,"不声明数据类型"
```

##### 多行写法

```go
var (
	a int = 100
    b bool = true
)
```
